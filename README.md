
# Automação do Processo de RFP Ágil

Este projeto apresenta uma solução desenvolvida utilizando Microsoft Power Platform (Power Apps, Power Automate e SharePoint) para automação do processo de contratação de terceiros.

## Tecnologias utilizadas
- Power Apps
- Power Automate
- SharePoint

## Funcionalidades
- Cadastro de solicitações
- Consulta de fornecedores
- Fluxo automatizado de aprovação
- Armazenamento em base corporativa

   ## Arquitetura da Solução
 
A solução foi desenvolvida utilizando a Microsoft Power Platform, com arquitetura baseada em três camadas:

- Power Apps: interface do usuário (front-end)
- Power Automate: lógica de negócio e automação de fluxos
- SharePoint: armazenamento de dados
